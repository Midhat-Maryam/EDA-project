# Customer & Sales Insights from Amazon Clothing Transactions

## 📌 Project Overview
This project explores customer behavior, sales performance, and return patterns in Amazon's clothing category. Using real-world messy data, the analysis applies **data cleaning, exploratory data analysis (EDA), and visualization** to uncover actionable business insights.

---

## 🎯 Learning Objectives
1. Cleaned and preprocessed real-world dataset (handle missing values, inconsistent types, and outliers).
2. Formulated and tested business-oriented hypotheses.
3. Performed **univariate, bivariate, and multivariate analysis**.
4. Derived insights about sales, returns, pricing, and customer demographics.
5. Communicated findings effectively through visualizations.
